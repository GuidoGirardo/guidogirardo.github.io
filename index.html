Guido Girardo's Android Portfolio

Personal Projects:

Study App: an app to track current and total study time, with a motivational quotes section.

Technologies: MVVM architecture, clean architecture, Kotlin, Jetpack Compose, API consumption with Retrofit and coroutines, unit testing, and UI testing.

Repository: https://github.com/GuidoGirardo/StudyApp








PlayStore Clone: a fully functional Play Store clone where users can register or log in, search for and download applications, and freely publish new apps.

Technologies: MVVM architecture using StateFlows, Kotlin, Jetpack Compose, Firebase (Firestore, Storage, Analytics), and data persistence with DataStore.

Repository: https://github.com/GuidoGirardo/playstore-2.0








Movies App: a movie and series streaming app with a built-in search engine and resume playback functionality. Users can continue watching content from where they left off, without needing to register an account.

Technologies: MVP architecture, Java, XML, ExoPlayer, Room, and data persistence with Room and SharedPreferences.

Repository: https://github.com/GuidoGirardo/darkmovies








Instagram UI: functional clone of the Instagram interface.

Technologies: Kotlin, Jetpack Compose, and reusable components.
